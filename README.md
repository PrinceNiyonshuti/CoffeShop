## Requirements

- Php V8 or greater
- create your .env file 
- create your database
- Xampp or apache server

## after run these script 

    composer install
    cp .env.example .env or copy .env.example .env
    php artisan key:generate
    php artisan migrate
    php artisan db:seed
    php artisan storage:link
